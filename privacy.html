Politique de confidentialité

Dernière mise à jour : 2 mai 2026

Votre vie privée est importante pour nous. Cette politique explique quelles données nous collectons et comment nous les utilisons.

1. Données collectées

Bible Verset du Jour collecte un minimum de données strictement nécessaires au fonctionnement du service :

  • Authentification TikTok : un jeton d'accès (token) pour publier en votre nom, stocké localement sur votre appareil
  • Identifiant public TikTok : votre open_id, fourni par TikTok lors de la connexion

Nous ne collectons aucune donnée personnelle sensible, aucun mot de passe, aucune information bancaire.

2. Utilisation des données

Les données collectées servent uniquement à :

  • Publier des vidéos de versets bibliques sur votre compte TikTok avec votre autorisation
  • Maintenir votre session d'authentification active

3. Stockage local

Toutes les données d'authentification sont stockées localement sur votre propre ordinateur, dans un fichier de configuration privé. Aucune donnée n'est transmise à des serveurs tiers en dehors de TikTok.

4. Partage des données

Nous ne vendons, ne louons ni ne partageons vos données avec qui que ce soit. Les seules communications de données se font directement entre votre appareil et l'API officielle de TikTok.

5. Services tiers

L'application utilise les services suivants :

  • TikTok API : pour la publication de contenu (soumis à la politique de TikTok)
  • Unsplash : pour les images (aucune donnée personnelle transmise)
  • Google Gemini : pour générer les commentaires (seul le texte du verset est envoyé)

6. Sécurité

Nous utilisons le protocole OAuth 2.0 sécurisé pour l'authentification. Les jetons d'accès sont stockés de manière sécurisée et peuvent être révoqués à tout moment depuis les paramètres de votre compte TikTok.

7. Vos droits

Vous pouvez à tout moment :

  • Révoquer l'accès de l'application depuis vos paramètres TikTok
  • Supprimer le fichier de configuration local contenant vos jetons
  • Cesser d'utiliser le service sans aucune conséquence

8. Données des mineurs

Ce service n'est pas destiné aux personnes de moins de 13 ans. Nous ne collectons pas sciemment de données concernant des mineurs.

9. Contact

Pour toute question concernant cette politique de confidentialité, contactez-nous via github.com/Devjam81
